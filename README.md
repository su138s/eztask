# eztask
지루한 반복 작업? 얘한테 시키고 커피 한잔하세요!
-> 일은 ‘eztas’가 하고 생색은 여러분이 내세요!

프로젝트 소개 (About)
‘eztask‘는 매일 똑같이 반복되는 지루하고 소모적인 행정·문서 작업을 단 몇 번의 클릭으로 끝내버리는 초간단 업무 자동화 툴킷입니다. 
귀찮은 단순 노동으로부터 여러분의 소중한 손목과 퇴근 시간을 지켜드립니다!!

개발 배경 (Why?)
- 복사·붙여넣기 지옥 탈출: 매번 엑셀이나 텍스트를 손으로 일일이 옮겨 적는 리소스 낭비를 막고 싶었습니다.
- 휴먼 에러(오탈자) 차단: 복붙하다가 발생하는 서식 깨짐, 오탈자, 데이터 누락 문제를 완벽하게 차단합니다.
- 삶의 질 향상: 수십 분 걸리던 노가다성 작업을 수 초 만에 끝내고 더 가치 있는 일에 집중하기 위해 만들었습니다.

주요 기능 (Features)
- 스마트 업무 정리: 사방으로 흩어져 있는 업무 리스트를 우선순위와 유형별로 깔끔하게 구조화합니다.
- 자동 파일 제너레이터: 가공 완료된 데이터를 바탕으로 실무에서 바로 쓰는 결과 파일(‘.csv’, ’.xlsx‘ 등)을 알아서 뚝딱 만들어 줍니다.
- 3초 스타터 가이드: 복잡한 설명서 필요 없이 미리 준비된 예시(Sample) 파일만 보고 따라 치면 바로 작동합니다.

사용 방법 (How to Use)
1. 눈으로 슥 보기: ’examples/‘ 폴더에 있는 초간단 예시 파일을 확인합니다.
2. 내용 쏙 넣기: 본인의 업무 서식이나 입력 데이터를 가이드 양식에 맞게 채워 넣습니다.
3. 결과 꿀꺽 하기: 프로그램을 실행하고 ’output/‘ 폴더에 완성된 마법 같은 결과 파일을 확인합니다.
   
앞으로의 계획 (Roadmap)
- 귀찮은 직장인/학생들을 위한 맞춤형 꿀 템플릿 대폭 추가 예정
- 검은 창(CLI)이 무서운 분들을 위해 마우스로 누르는 화면(GUI) 지원 예정
- 이메일 자동 발송 및 외부 API 연동 기능 확장 예정
